Crystal Creator

What is it?

Crystal Creator is an interactive web application based off of Matthew Epler's YouTube series on Designing Generative Systems with p5.js (with an extra shout out to Stephan Max for more inspiration). It creates geometric "crystals" from a set of randomly generated parameters.

How to use it?

This page loads with a new set of crystals every time. If you see one you like, click to download or edit it. In edit mode, you can tweak it to your liking, or create a new one of your own!

Why?

Crystal Creator was initially created in 2020 and served as a creative outlet to help combat shelter-in-place boredom and depression.
